# The Universal Graph Machine

**A machine that reasons — and can always show its work.**

Most computers are oracles. You ask, they answer, and you just have to trust
them. The Universal Graph Machine is different: it works things out from what it
knows, and whenever you want, it can tell you *exactly why* it reached an
answer — or honestly admit that it doesn't know.

This is a book for the curious. You don't need to be a programmer. If you can
follow a detective piecing together a case, you can follow this machine
thinking.
